# 01 - Active Directory Lab

## Opis
Praktyczne laboratorium z konfiguracji Active Directory na Windows Server 2022.
Obejmuje instalację, konfigurację sieciową, strukturę OU, GPO oraz delegowanie uprawnień.

## Wyzwanie
1. Stworzenie nowej maszyny z Windows Server 2022
2. Stworzenie odpowiedniej konfiguracji sieciowej
3. Stworzenie odpowiednich Jednostek Organizacyjnych, dodanie członków oraz oddelegowanie zadań takich jak zmiana haseł działowi IT
4. Stworzenie i konfiguracja zasad grupy (GPO) — polityka haseł, blokada ekranu, ograniczenia dla użytkowników
5. Stworzenie grup bezpieczeństwa (Security Groups) i przypisanie ich do odpowiednich jednostek organizacyjnych
6. Delegowanie dodatkowych uprawnień dla działu IT — odblokowywanie kont, tworzenie użytkowników
7. Konfiguracja Fine-Grained Password Policy — różne polityki haseł dla różnych grup
8. Włączenie Active Directory Recycle Bin — przywracanie usuniętych obiektów
9. Konfiguracja Audit Policy — monitorowanie logowań oraz zmian w strukturze AD

## Środowisko
| Parametr | Wartość |
|----------|---------|
| System | Windows Server 2022 |
| Rola | Active Directory Domain Services |
| Domena | corp.local |
| Platforma | VMware / VirtualBox |
