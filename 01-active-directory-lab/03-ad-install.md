## ****Instalowanie Active Directory w Microsoft Server***

-> Na początku wchodzimy w DASHBOARD w Microsoft Server

<img width="730" height="352" alt="image" src="https://github.com/user-attachments/assets/a23e3e54-0fcd-4a8b-99d4-cdf95553cc30" />

-> Klikamy opcję "Add roles and features, powinno się pojawić następujące okienko:

<img width="783" height="557" alt="image" src="https://github.com/user-attachments/assets/73a199c2-3610-48de-936c-ce79a90570e9" />

-> Następnie w zakładce Before you Begin po zapoznaniu klikamy "NEXT"
-> Przy zakładce Installation Type wybieramy opcje Role-Based or Feature-based installation
-> Kolejno w zakładce Serwer Selection wybieramy opcję:
  -> Select a server from the server pool

-> Następnie w Servers Role wybieramy, to co jest zaznaczone na obrazku poniżej:

<img width="787" height="557" alt="image" src="https://github.com/user-attachments/assets/c776d40a-8e7a-4271-b60a-f346a5fd8f96" />

-> Przez kolejne zakładki przechodzimy dalej klikamy "Next"
-> Na ostatniej klikamy tylko zamiast Next "Install", obrazek poniżej:

<img width="780" height="558" alt="image" src="https://github.com/user-attachments/assets/b91f47a7-ee72-4f5d-9e4a-9ac92107c789" />
